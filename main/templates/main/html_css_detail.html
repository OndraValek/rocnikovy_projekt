Jazyk HTML a jeho verze, HTML syntax a validita kódu

Co je HTML?

HTML (HyperText Markup Language) je jazyk určený pro tvorbu webových stránek.

Značky (tagy)

Používá značky (tagy) k popisu obsahu stránky

Tvůrce

První verzi vytvořil Tim Berners-Lee v roce 1991

Vývoj verzí HTML

HTML 4.01

Široce používaná verze, vycházející z SGML

XHTML

Verze vycházející z XML, vyžadovala striktní pravidla syntaxe

HTML5

Současná verze, přináší nové sémantické prvky, podporu multimédií a API

HTML syntaxe

HTML dokument je tvořen elementy, které jsou obaleny značkami (tagy).

Tagy

Tagy se píšou do ostrých závorek: <tag>obsah</tag>

Párové tagy: <p>...</p>
Nepárové tagy: <img src="img.jpg" alt="obrázek">

Atributy

Poskytují dodatečné informace: <img src="img.jpg" alt="obrázek">

Zapisují se jako název="hodnota" vždy v uvozovkách
Mohou být volitelné nebo povinné

Komentáře

Komentáře se píšou mezi <!-- a -->: <!-- Komentář -->

Entity

Používají se pro zápis speciálních znaků: &lt; (<), &gt; (>)

Validita HTML kódu

W3C Validátor

Validitu HTML lze ověřit pomocí W3C Validátoru: validator.w3.org

Problémy s nevalidním kódem

Nevalidní kód může způsobit problémy s zobrazením stránky a indexováním ve vyhledávačích

Struktura webové stránky, základní HTML elementy

Základní struktura HTML stránky

DOCTYPE

Dokument HTML začíná deklarací <!DOCTYPE html>, který určuje verzi HTML a využívá se pro validaci kódu

Kořenový prvek

Kořenovým prvkem je <html>, obsahuje hlavičku <head> a tělo <body>

Hlavička

Hlavička obsahuje meta informace (kódování, popis stránky) a odkazy na CSS nebo JavaScript

Tělo

Tělo obsahuje obsah stránky (nadpisy, odstavce, obrázky, odkazy)

Základní HTML elementy

Blokové elementy

Zobrazují se na novém řádku, tvoří bloky obsahu:

Nadpisy: <h1> ... <h6>
Odstavce: <p>
Oddíly: <div>
Seznamy: <ul>, <ol>, <li>
Tabulky: <table>, <tr>, <td>, <th>

Inline elementy

Zobrazují se vedle sebe a nevytváří bloky:

Úpravy textu: <span>, <strong>, <em>, <b>, <i>
Obrázky: <img src="img.jpg" alt="Popis">
Zalomení řádku: <br>

Sémantické elementy

Určeny k popisu struktury stránky:

Záhlaví: <header>
Navigace: <nav>
Obsah: <main>, <article>, <section>
Patička: <footer>

Odkazy a navigace mezi stránkami

Hypertextové odkazy

Odkazy se vytvářejí pomocí značky <a>: <a href="https://www.example.com" target="_blank">Odkaz</a>

Atribut href

Určuje cílovou adresu odkazu

Atribut target

target="_blank" otevře odkaz v novém okně

Typy odkazů

Absolutní odkaz

Obsahuje celou URL adresu (např. https://www.example.com)

Relativní odkaz

Odkazuje na soubor v rámci stejného webu (např. index.html)

Kotva

Odkaz na konkrétní část stránky: <a href="#section">Odkaz na sekci</a>, <h2 id="section">Sekce</h2>

Navigační prvky

Element nav

Navigace bývá umístěna v elementu <nav>

Seznamy

Pro navigaci se často používají seznamy, např. <ul> a <li>

Umístění navigace

Hlavní navigace bývá obvykle v záhlaví stránky, může být horizontální nebo vertikální, její součástí bývají rozbalovací menu, logo a vyhledávací pole

Breadcrumb

Drobečková navigace, zobrazuje cestu k aktuální stránce, např. Domů > Kategorie > Podkategorie > Stránka

Mobilní navigace

Navigace na mobilních zařízeních bývá často skrytá za tlačítkem (tzv. hamburger menu)

Vlastnosti navigace

Důležitými vlastnostmi navigace jsou jednoduchost, přehlednost a dostupnost

Kaskádové styly a integrace s HTML, syntax CSS, typy selektorů

Kaskádové styly

CSS (Cascading Style Sheets) je jazyk pro formátování HTML dokumentů.

Oddělení obsahu od vzhledu

Umožňuje oddělit obsah od vzhledu a definovat styly pro různá zařízení

Kaskádování

Styly se aplikují shora dolů (představa průchodu kaskádou) a mohou být děděny i přepsány

Standardy CSS

Standardy CSS jsou definovány organizací W3C, aktuální verze je CSS3

Soubory CSS

Soubory CSS mají obvykle příponu .css a odkazují se v hlavičce HTML dokumentu

Struktura CSS

Základem CSS jsou tzv. pravidla (rules):

Selektor

Určuje, na které elementy se pravidlo vztahuje

Vlastnost

Popisuje, jaká vlastnost se mění (např. color, font-size)

Hodnota

Konkrétní hodnota vlastnosti (např. red, 20px)

Způsoby integrace CSS

Inline styly

Přímo v HTML elementu; používají se zřídka:

<p style="color: blue; font-size: 20px;">Modrý text</p>

Interní styly

V <style> uvnitř <head>:

<style>
p { color: blue; font-size: 20px; }
</style>

Externí styly

V samostatném souboru .css:

<link rel="stylesheet" href="styles.css">

Typy selektorů

Selektor elementu

h1 { color: red; }

Selektor třídy

.box { border: 1px solid black; }

Selektor ID

#menu { background-color: grey; }

Kontextový selektor

div p { font-size: 16px; }

Hromadný selektor

h1, h2, h3 { color: blue; }

Pseudo-třídy

a:hover { text-decoration: underline; }

Layout webové stránky a zásady webdesignu

Základy layoutu webové stránky

Webový layout určuje, jak budou prvky stránky uspořádány.

Tabulky (zastaralé)

Dříve byl layout tvořen tabulkami, dnes se používají CSS grid a flexbox

Flexbox

Ideální pro jednorozměrné rozložení prvků (řádky nebo sloupce); umožňuje jednoduché zarovnání a řazení prvků buď vedle sebe nebo pod sebou

CSS Grid

Dvourozměrný layout systém, který umožňuje vytvářet složitější struktury; je vhodný pro rozložení celé stránky nebo bloků obsahu

Responsivní design

Responsivní design zajišťuje, že se stránka přizpůsobí různým zařízením (mobil, tablet, desktop).

Media queries

Pro responsivní design se používají media queries, které umožňují definovat styly pro různé šířky obrazovky

Viewport

Viewport je oblast, kterou prohlížeč zobrazuje; je důležité nastavit správnou šířku viewportu pro mobilní zařízení

Relativní velikosti

Velikost textu a obrázků by měla být relativní (v procentech nebo em) a ne absolutní (v pixelech)

Zásady webdesignu

Přehlednost

Snadná orientace a čitelnost, vhodné zarovnání a odstupy, omezení počtu prvků

Konzistence

Stejné styly a pravidla pro všechny stránky, jednotný design

Dostupnost

Respektování standardů pro zrakově postižené, použití alternativních textů u obrázků, kontrastní barvy

Responsivita

Optimalizace pro různá zařízení

Grafika

Kvalitní obrázky, ikony, grafy, vhodné formáty, optimalizace pro rychlost načítání

Typografie

Čitelný text, vhodné fonty (omezit počet), kontrastní barvy
