Seznam témat

Úvod do programování

Základy

Programování je proces vytváření počítačových programů pomocí programovacích jazyků. Zahrnuje analýzu problému, návrh řešení, implementaci kódu a testování. Moderní programování se zaměřuje na čitelnost, udržitelnost a efektivitu kódu.

Datové struktury

Algoritmy

Datové struktury jsou způsoby organizace a ukládání dat v počítači tak, aby s nimi bylo možné efektivně pracovat. Mezi základní datové struktury patří pole, seznamy, zásobníky, fronty, stromy a grafy.

Webové technologie

Frontend

Moderní webové aplikace využívají kombinaci HTML, CSS a JavaScriptu pro vytváření interaktivních uživatelských rozhraní. Backend technologie jako Django, Flask nebo Node.js zajišťují serverovou logiku a databázové operace.

Databáze a SQL

Backend

Databáze jsou systémy pro ukládání, organizaci a správu dat. SQL (Structured Query Language) je standardní jazyk pro práci s relačními databázemi. Umožňuje vytváření, čtení, aktualizaci a mazání dat pomocí různých příkazů.

Objektově orientované programování

OOP

OOP je programovací paradigma založené na konceptu objektů, které obsahují data (atributy) a kód (metody). Hlavní principy OOP zahrnují zapouzdření, dědičnost, polymorfismus a abstrakci, což umožňuje vytváření modulárního a znovupoužitelného kódu.

Bezpečnost aplikací

Security

Bezpečnost je kritickým aspektem vývoje aplikací. Zahrnuje ochranu před běžnými útoky jako SQL injection, XSS, CSRF a dalšími. Důležité je implementovat autentizaci, autorizaci, šifrování dat a pravidelné bezpečnostní aktualizace.
