Základy informatiky

1

Informatika je vědní obor zabývající se získáváním, zpracováním, ukládáním a přenosem informací. Zahrnuje teoretické i praktické aspekty výpočetní techniky, programování, algoritmů a databází. Zkoumá pojmy jako informace, data, signály, digitalizace a číselné soustavy.

Programy a data

2

Programy a data zahrnují programové a datové soubory, systémové a aplikační programy, utility, softwarové licence, instalaci a zálohování programů a dat, cloudová řešení, emulaci a virtualizaci. Pokrývá veškeré aspekty práce se softwarem.

Informační systémy a databázové systémy

3

Informační systémy jsou soubory technologií, pravidel a dat sloužících ke sběru, zpracování, ukládání a distribuci informací. Pokrývá komponenty IS, základní typy systémů (TPS, MIS, DSS), ERP, CRM, GIS a životní cyklus vývoje IS.

Rastrová grafika a digitální fotografie

4

Rastrová grafika je grafický formát tvořený mřížkou pixelů. Zahrnuje principy rastrové grafiky, vlastnosti obrazu, rastrové formáty, vnímání barev, barevné modely (RGB, CMYK, HSB), funkce rastrových editorů a digitální fotografii s expozicí a zpracováním.

Vektorová grafika

5

Vektorová grafika využívá matematické rovnice pro definici tvarů místo pixelů. Zahrnuje principy vektorové grafiky, vektorizaci a rasterizaci, oblasti využití, vektorové formáty, pomůcky pro přesné kreslení, manipulace s křivkami, CAD, 3D modelování, VR a AR.

Zpracování textů na počítači

6

Zpracování textů zahrnuje programy pro práci s textem (editory, procesory, DTP), formáty textových dokumentů, typografii, typy fontů, strukturu dokumentu, styly a šablony, pokročilé nástroje MS Word, předtiskovou přípravu a elektronické knihy.

Počítačové zpracování zvuku

8

Počítačové zpracování zvuku zahrnuje vlastnosti zvuku a jeho digitalizaci, kompresi zvuku pomocí psychoakustických modelů, audio kodeky a formáty, zvukovou syntézu a MIDI, tvorbu digitální hudby, programy pro editaci zvuku a využití umělé inteligence při práci se zvukem.

Digitální video a multimediální prezentace

9

Digitální video a multimediální prezentace zahrnují principy analogového a digitálního videa, technické vybavení, parametry digitálního videa, kodeky a formáty videa, proces produkce a postprodukce videa, zásady pořizování záběrů, typy záběrů, DVD authoring, publikování videa na internetu a počítačovou animaci s využitím AI.

Relační databáze a SQL

10

Relační databáze a SQL zahrnují princip relační databáze, relační datový model, typy klíčů a propojení tabulek, proces návrhu databáze, ER diagramy, normalizaci databáze, správu databáze, integritu a indexaci, jazyk SQL, základní SQL dotazy a pokročilé možnosti práce s databází (pohledy, triggery, uložené procedury a funkce).

Internet a WWW

11

Internet a WWW zahrnují principy a vývoj internetu, adresování (IP adresy, DNS, domény), principy a standardy WWW, URI (URL), protokol HTTP/HTTPS s metodami a stavovými kódy, funkce internetového prohlížeče, sessions, cookies a local storage, bezpečnostní aspekty, vyhledávače, SEO, analytiku webu a trendy vývoje webu (Web 2.0, Web 3.0, IoT).

HTML a kaskádové styly

12

HTML a kaskádové styly zahrnují jazyk HTML a jeho verze (HTML 4.01, XHTML, HTML5), HTML syntax a validitu kódu, strukturu webové stránky, základní HTML elementy (blokové, inline, sémantické), odkazy a navigaci mezi stránkami, kaskádové styly (CSS), integraci CSS s HTML, syntax CSS, typy selektorů, layout webové stránky (Flexbox, CSS Grid) a zásady webdesignu (responsivní design, přehlednost, konzistence, dostupnost).

Webové technologie

13

Webové technologie zahrnují model klient-server, frontend a backend, klientské technologie (HTML, CSS, JavaScript, knihovny a frameworky), serverové technologie (HTTP servery, Node.js, PHP, Python, Java, C#, databáze), základy JavaScriptu (syntax, datové typy, DOM, BOM), tvorbu serverových aplikací s Node.js (synchronní a asynchronní programování), webová API (REST, GraphQL, WebSockets), AJAX a komunikaci mezi frontendem a backendem, webové knihovny, frameworky a nástroje pro vývoj.
